AnoDomino Inventory Management System is an enterprise-grade stock and asset control solution developed for AnoDomino Company. This specialized system handles complex inventory workflows, stock movements, and financial reporting with precision, built on the robust Laravel framework.

The system provides:

- **Real-time Inventory Dashboard** - At-a-glance view of stock levels, recent activities, and system health
- **Advanced Stock Management** - Support for SKU tracking, categories, and multi-warehouse locations
- **Automated Alerts** - Low-stock notifications and expiration date tracking to prevent operational delays
- **Role-Based Access (RBAC)** - Granular permissions for Admins, Managers, and Staff members
- **Activity Logging** - Comprehensive audit trails for every transaction
